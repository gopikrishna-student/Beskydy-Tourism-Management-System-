## 📌 Project Overview

The Beskydy Tourism Management System is a Command Line Interface (CLI) based interactive application developed using the Janet programming language.

This system simulates a tourism information platform where users can explore attractions, accommodation options, seasonal activities, and contact information related to the Beskydy region.

The project demonstrates structured programming, user interaction, input validation, and loop control mechanisms.

---

## 🎯 Objectives

- Design a menu-driven CLI application
- Implement structured programming concepts
- Apply input validation techniques
- Simulate a real-world tourism management system
- Demonstrate practical use of Janet programming language

## 📊 System Flow

1. Program starts using main function.
2. Menu options are displayed.
3. User selects an option (1–5).
4. Input is validated.
5. Corresponding information is displayed.
6. Program continues until Exit is selected.
